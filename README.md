🏨 La Maison DTN – Luxury Hotel Management & Booking System

La Maison DTN is a luxury hotel booking and management platform designed for a 5-star European-style hotel located in Vietnam.
The system allows customers to explore rooms, make reservations, and manage bookings while providing administrators with powerful tools to manage rooms, guests, and hotel operations.

The platform combines a modern ReactJS frontend with a robust Laravel backend API, delivering a smooth, scalable, and secure hotel management experience.

Features
👤 Customer Features

User registration and login

Personal profile management

View and update personal information

Browse hotel rooms using an interactive Room Map

Book rooms easily with check-in / check-out dates

View current reservations

View booking history

Download booking information
